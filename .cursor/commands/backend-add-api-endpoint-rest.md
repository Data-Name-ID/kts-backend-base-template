# Добавление или изменение REST API endpoint

Используй skill **backend-add-api-endpoint-rest** (`.cursor/skills/backend-add-api-endpoint-rest/SKILL.md`).

## Что сделать

Выполни workflow skill: зафиксируй контракт endpoint, соблюдай REST-семантику, реализуй endpoint через DI и существующие слои, опиши error-responses в OpenAPI, подключи роутинг и добавь тесты.
